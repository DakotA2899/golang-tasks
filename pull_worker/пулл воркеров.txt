Суть задачи
1.	У вас есть массив задач: [1, 2, 3, ..., 20]. Каждая задача — это число, для которого нужно вычислить квадрат.
2.	Вместо того чтобы создавать новую горутину для каждой задачи, используйте фиксированное количество воркеров (например, 4, но они не должны запускаться все сразу, они запускаются только тогда когда поступает задача и некому ее обработать(использовать select{})), которые переиспользуются.
3.	Задачи поступают в пул, и воркеры берут их из очереди на обработку.
4.	После завершения всех задач программа должна корректно завершить работу всех воркеров.
5.	Использовать отмену реализованную в первой задаче
